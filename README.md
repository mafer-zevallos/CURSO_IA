# SALDO FINO

Mockup inicial en React para una app web de control de gastos personales, basado en `plan_app_control_gastos.md`.

## Stack

- React
- Vite

## Pantallas del mockup

- Resumen
- Gastos
- Nuevo gasto
- Categorias
- Reportes

## Para correrlo

1. Instala dependencias con `npm install`
2. Inicia el entorno con `npm run dev`
